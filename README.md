# sim-internal-beehive

System Information Management Internal Beehive.

# Requiments:

- Node.JS

# Installation:

Dev Enviroments:

```cmd
$ cd sim-internal-beehive
$ npm install
$ npm run dev
```

Production Enviroments:

```cmd
$ cd sim-internal-beehive
$ npm install
$ npm start
```

Run on

```cmd
localhost:3005
```
